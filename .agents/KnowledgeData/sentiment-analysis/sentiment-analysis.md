## 一、市場情緒指標

### 1. 恐懼貪婪指數（Fear & Greed Index）

**來源：** alternative.me

**組成：**
| 因素 | 權重 | 說明 |
|------|------|------|
| 波動率 | 25% | 當前波動率 vs 歷史 |
| 成交量 | 25% | 當前成交量 vs 歷史 |
| 社交媒體 | 15% | Twitter/Reddit 情緒 |
| 調查 | 15% | 投資者調查 |
| 主導地位 | 10% | BTC 市值佔比 |
| 趨勢 | 10% | Google 搜索趨勢 |

**解讀：**
| 指數 | 情緒 | 策略 | 驗證 |
|------|------|------|------|
| 0-20 | 極度恐懼 | 買入機會 | ✅ |
| 20-40 | 恐懼 | 考慮買入 | ✅ |
| 40-60 | 中性 | 持有/觀望 | ✅ |
| 60-80 | 貪婪 | 考慮賣出 | ✅ |
| 80-100 | 極度貪婪 | 賣出機會 | ✅ |

**歷史驗證：**
- 2020-03 COVID：指數 8 → 隨後漲 1000%
- 2021-11 頂部：指數 75-80 → 隨後跌 75%
- 2022-06 LUNA：指數 10 → 隨後觸底

**交易策略：**
```
極度恐懼（<20）：
- 開始分批買入
- 設定寬止損（-20%）
- 持有 3-6 個月

極度貪婪（>80）：
- 開始分批賣出
- 減少槓桿
- 保留現金
```

**驗證**：✅ 長期驗證有效

### 2. 資金費率（Funding Rate）

**定義：** 永續合約中，多空雙方定期支付的費用

```
資金費率 > 0：多頭支付空頭 → 市場看多
資金費率 < 0：空頭支付多頭 → 市場看空
```

**解讀：**
| 費率 | 含義 | 策略 |
|------|------|------|
| > 0.1%（8h） | 極度看多 | 考慮做空 |
| 0.01-0.1% | 偏多 | 正常 |
| -0.01-0.01% | 中性 | 正常 |
| -0.1--0.01% | 偏空 | 正常 |
| < -0.1%（8h） | 極度看空 | 考慮做多 |

**極端資金費率交易：**
```
費率 > 0.1%：
- 市場過度看多
- 考慮做空
- 收取資金費率

費率 < -0.1%：
- 市場過度看空
- 考慮做多
- 收取資金費率
```

**驗證**：✅ 多次驗證

### 3. 未平倉合約（Open Interest, OI）

```
OI 上升 + 價格上升 = 多頭強勢（趨勢延續）
OI 上升 + 價格下跌 = 空頭強勢（趨勢延續）
OI 下降 + 價格上升 = 空頭回補（可能反轉）
OI 下降 + 價格下跌 = 多頭平倉（可能觸底）
```

**解讀：**
| OI 變化 | 價格變化 | 含義 | 策略 |
|---------|---------|------|------|
| ↑ | ↑ | 多頭強勢 | 持多或加多 |
| ↑ | ↓ | 空頭強勢 | 持空或加空 |
| ↓ | ↑ | 空頭回補 | 多頭可能到頂 |
| ↓ | ↓ | 多頭平倉 | 可能觸底 |

---

### 4. 交易所流入/流出

```
交易所流入增加 → 準備賣出 → 看空
交易所流出增加 → 準備持有 → 看多
```

## 三、鏈上情緒指標

### 7. MVRV Z-Score

```
MVRV = 市值 / 已實現市值
Z-Score = (MVRV - 平均MVRV) / 標準差

解讀：
Z > 7：市場過熱（賣出）
Z < 0：市場低估（買入）
```

**歷史驗證：**
- 2021-11：Z > 7 → 頂部
- 2022-12：Z < 0 → 底部
- 2024-03：Z > 5 → 短期回調

---

### 8. NUPL（Net Unrealized Profit/Loss）

```
NUPL > 0.75：大量未實現利潤 → 可能見頂
NUPL < 0：大量未實現虧損 → 可能見底
```

---

### 9. SOPR（Spent Output Profit Ratio）

```
SOPR > 1：賣出者獲利 → 多頭主導
SOPR < 1：賣出者虧損 → 空頭主導
SOPR 持續 < 1：投降階段 → 可能觸底
```

## 四、市場週期心理

### 10. 市場心理循環

```
1. 懷疑（Disbelief）
   - 價格開始上漲
   - 多數人認為只是反彈
   - 最佳買入時機

2. 樂觀（Optimism）
   - 價格持續上漲
   - 開始相信牛市
   - 可以持有

3. 興奮（Excitement）
   - 價格快速上漲
   - FOMO 情緒蔓延
   - 開始考慮賣出

4. 貪婪（Greed）
   - 價格創新高
   - 所有人都在談論
   - 開始分批賣出

5. 否認（Denial）
   - 價格開始下跌
   - 認為只是回調
   - 繼續持有

6. 恐懼（Fear）
   - 價格持續下跌
   - 開始恐慌
   - 考慮止損

7. 恐慌（Panic）
   - 價格暴跌
   - 大量拋售
   - 最佳買入時機（但需要勇氣）

8. 投降（Capitulation）
   - 價格觸底
   - 多數人放棄
   - 開始分批買入

9. 憤怒（Anger）
   - 價格低迷
   - 對市場失去信心
   - 繼續買入

10. 沮喪（Depression）
    - 價格橫盤
    - 失去興趣
    - 最後買入機會
```

## 五、情緒交易策略

### 11. 反向指標策略

```
當所有人都看多 → 準備賣出
當所有人都看空 → 準備買入

具體指標：
- 恐懼貪婪 > 80 → 開始賣出
- 恐懼貪婪 < 20 → 開始買入
- 資金費率 > 0.1% → 考慮做空
- 資金費率 < -0.1% → 考慮做多
- 社交媒體極度樂觀 → 減少倉位
- 社交媒體極度悲觀 → 增加倉位
```

---

### 12. 確認策略

```
不要只用情緒指標做決策
需要配合：
1. 技術分析確認
2. 基本面確認
3. 資金管理

情緒指標最佳用途：
- 確認趨勢強度
- 識別極端情況
- 輔助入場/出場時機
```
