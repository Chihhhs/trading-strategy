# Market Structure Shift（MSS）完整指南

> 更新：2026-06-16
> 案例：SOL 即時 $72.01，90d 下降趨勢，尚未出現 MSS

---

## 一、什麼是 Market Structure Shift？

**MSS = 市場結構轉換 = 趨勢方向的根本改變**

```
下降趨勢中：價格持續創新低（LH + LL）
突然：價格突破前高（HH）→ MSS 確認 → 趨勢轉為上升

上升趨勢中：價格持續創新高（HH + HL）
突然：價格跌破前低（LL）→ MSS 確認 → 趨勢轉為下降

MSS 是「趨勢反轉」的最可靠信號之一
比 RSI、MACD 等指標更直接、更不容易有假信號
```

---

## 二、市場結構的基礎：HH / HL / LH / LL

### 上升趨勢（Bullish Structure）

```
        H3 ← Higher High（更高高點）
       /
      H2 ← Higher High
     /
    H1
   /
  L1
   \
    L2 ← Higher Low（更高低點）
     \
      L3 ← Higher Low

HH + HL = 上升趨勢確認
每個高點都比前高更高
每個低點都比前低更高
```

### 下降趨勢（Bearish Structure）

```
  H1
   \
    H2 ← Lower High（更低高點）
     \
      H3 ← Lower High
       /
      L1
       \
        L2 ← Lower Low（更低低點）
         \
          L3 ← Lower Low

LH + LL = 下降趨勢確認
每個高點都比前高更低
每個低點都比前低更低
```

### 盤整（Consolidation）

```
HH + LL = 收斂三角形（準備突破）
LH + HL = 擴張三角形（波動加大）

盤整結束後，價格會選擇方向突破
```

## 三、MSS 的兩種類型

### MSS 看多（下降 → 上升）

```
步驟 1：下降趨勢中（LH + LL）
  H1 $75 → H2 $74 → H3 $73（LH）
  L1 $72 → L2 $71 → L3 $70（LL）

步驟 2：價格止跌，開始反彈
  價格從 $70 反彈到 $73.5
  → 突破前高 H3 $73 ✅

步驟 3：MSS 確認！
  價格突破前高 = 下降慣性被打破
  → 趨勢可能轉為上升

步驟 4：HL 確認（Higher Low）
  價格回調，但不破前低 $70
  → 在 $71 形成 HL ✅
  → 上升趨勢確認！
```

### MSS 看空（上升 → 下降）

```
步驟 1：上升趨勢中（HH + HL）
  H1 $70 → H2 $72 → H3 $74（HH）
  L1 $68 → L2 $69 → L3 $70（HL）

步驟 2：價格回落，開始下跌
  價格從 $74 跌到 $70.5
  → 跌破前低 L3 $70 ✅

步驟 3：MSS 確認！
  價格跌破前低 = 上升慣性被打破
  → 趨勢可能轉為下降

步驟 4：LH 確認（Lower High）
  價格反彈，但不過前高 $74
  → 在 $73 形成 LH ✅
  → 下降趨勢確認！
```

## 四、MSS 的關鍵要素

### 要素 1：突破 / 跌破必須「有效」

```
❌ 假突破：價格突破前高後立刻回落
✅ 有效突破：價格收盤站穩前高之上

判斷標準：
- 日線收盤價 > 前高 = 有效突破
- 日線收盤價 < 前低 = 有效跌破
- 盤中突破但收盤回落 = 假突破
```

### 要素 2：回測確認（Retest）

```
MSS 後，價格通常會回測突破位：

看多 MSS：突破前高 → 回測前高（變支撐）→ 不破 → HL 確認
看空 MSS：跌破前低 → 回測前低（變阻力）→ 不過 → LH 確認
```

### 要素 3：成交量確認

```
✅ 突破時放量 = 有效突破
❌ 突破時縮量 = 可能假突破
```

### 要素 4：時間框架共振

```
最強 MSS：多個時間框架同時出現 MSS
- 日線 MSS + 週線 MSS = 最強信號
- 只有日線 MSS = 中等信號
- 只有 1H MSS = 弱信號
```

## 五、MSS 與其他指標的配合

### MSS + RSI

```
MSS 看多 + RSI 從超賣區反彈 = 最佳做多信號
MSS 看多 + RSI 已經在超買區 = 可能已經太晚
MSS 看空 + RSI 從超買區回落 = 最佳做空信號
```

### MSS + 費波那契

```
MSS 看多 + 價格在 38.2% 回撤位 = 最佳做多入場
MSS 看空 + 價格反彈到 38.2% = 最佳做空入場
```

### MSS + 成交量

```
MSS 看多 + 突破時放量 = 確認
MSS 看多 + 突破時縮量 = 可能是假突破
```

## 六、SOL 實戰分析

### 當前狀態（2026-06-16）

```
SOL 即時: $72.01

波段結構：
  高點: $75.60 → $75.30 → $74.13 → $73.84（LH + LH + LH）
  低點: $73.75 → $73.39 → $72.93 → $72.25 → $72.01（LL + LL + LL + LL）

趨勢：LH + LL = 下降趨勢
MSS：❌ 未出現（價格沒有突破任何前高）
HL：❌ 沒有（每個低點都比前低更低）
```

### MSS 看多需要什麼？

```
條件 1：價格突破最近前高 $73.84
  → 目前 $72.01，距離 -2.5%
  → 需要漲 +2.5% 才能觸發 MSS

條件 2：突破後回測不破 $73.84（前高變支撐）
  → 如果回調到 $73.84 以上 → HL 確認

條件 3：成交量確認
  → 突破時需要放量

→ 如果 SOL 漲到 $74 以上 → 考慮加倉
→ 如果 SOL 繼續跌 → 不要加倉
```

### MSS 看空需要什麼？

```
條件 1：價格跌破最近前低 $72.01
  → 如果繼續跌 → 下降趨勢加速

條件 2：反彈不過 $72.01（前低變阻力）
  → 如果反彈到 $72 附近被拒絕 → LH 確認

→ 如果 SOL 跌破 $72 → 你的多倉風險更高
→ 如果 SOL 反彈到 $72 被拒絕 → 考慮做空
```

## 七、MSS 的常見陷阱

| 陷阱 | 後果 | 正確做法 |
|------|------|---------|
| 太早判斷 MSS | 接刀子 | 等突破 + 回測確認 |
| 忽略假突破 | 被套 | 日線收盤價確認 |
| 忽略時間框架 | 噪音交易 | 至少確認日線 + 4H |
| 忽略成交量 | 假信號 | 突破時需要放量 |
| 逆勢操作 | 虧損 | 等 MSS 確認再順勢 |

## 八、MSS 的檢查清單

```
每次想入場前，問自己：

[ ] 1. 長期趨勢是什麼？（HH+HL 或 LH+LL）
[ ] 2. 有沒有 MSS？（突破前高或跌破前低）
[ ] 3. MSS 有沒有回測確認？（HL 或 LH）
[ ] 4. 成交量有沒有確認？
[ ] 5. 多個時間框架有沒有共振？
[ ] 6. RSI 支持嗎？
[ ] 7. 費波那契位支持嗎？

如果 5/7 以上確認 → 入場
如果 3/7 → 小倉位
如果 < 3/7 → 不入场
```
