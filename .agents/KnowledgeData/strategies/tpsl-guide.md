# TP/SL 設定完整指南

> 更新：2026-06-16
> 核心原則：SL 必須有技術依據，不能隨便設

---

## 一、SL（止損）的六種設定方式

### 方法 1：ATR 止損（推薦）

```
公式：SL = 入場價 ± N × ATR(14)

N = 1.5 ~ 2.0（推薦 2.0）
ATR = 14 天平均真實波幅

為什麼用 ATR：
- ATR 反映市場波動率
- 波動大的幣，SL 要設寬一點
- 波動小的幣，SL 可以設緊一點

範例（HYPE 做多 @ $71.50）：
  ATR(14) = $0.91
  1.5x ATR = $1.37 → SL = $71.50 - $1.37 = $70.13
  2.0x ATR = $1.82 → SL = $71.50 - $1.82 = $69.68  ← 推薦
  3.0x ATR = $2.73 → SL = $71.50 - $2.73 = $68.77

範例（SOL 做多 @ $73.14）：
  ATR(14) ≈ $0.41
  2.0x ATR = $0.82 → SL = $73.14 - $0.82 = $72.32  ← 推薦
```

**優點：** 根據市場波動率自適應
**缺點：** 需要計算 ATR

### 方法 2：波段高低點止損

```
公式：
  做多 SL = 最近波段低點 - 緩衝（0.5x ATR 或 1%）
  做空 SL = 最近波段高點 + 緩衝

為什麼用波段高低點：
- 波段低點 = 市場認可的支撐位
- 跌破波段低點 = 趨勢可能反轉
- 技術分析中最直觀的止損位

範例（HYPE 做多 @ $71.50）：
  最近波段低點 = $72.87（14天前）→ 但比入場高，不適用
  次近波段低點 = $72.49（19天前）→ 也比入場高
  再下一個 = $66.72（40天前）→ 太遠
  → 這種情況下，用 ATR 止損更合適

範例（SOL 做多 @ $73.14）：
  最近波段低點 = $72.25（4天前）
  SL = $72.25 - 1% = $71.53
  或 SL = $72.25 - 0.5x ATR = $72.25 - $0.21 = $72.04
```

**優點：** 直觀，有技術依據
**缺點：** 有時波段低點比入場價還高（上升趨勢中）

### 方法 3：EMA 止損

```
公式：
  做多 SL = EMA50 下方（或 EMA20 下方）
  做空 SL = EMA50 上方（或 EMA20 上方）

為什麼用 EMA：
- EMA 是動態支撐/阻力
- 價格跌破 EMA50 = 中期趨勢可能反轉
- 比固定比例更靈活

範例（HYPE 做多 @ $71.50）：
  EMA50 = $71.57 → 價格在 EMA50 下方，不適用
  EMA20 = $73.10 → 價格在 EMA20 下方，不適用
  → 這種情況下，用 ATR 止損

範例（SOL 做多 @ $73.14）：
  EMA50 = $73.78 → 價格在 EMA50 下方，不適用
  EMA20 = $73.96 → 價格在 EMA20 下方，不適用
  → 這種情況下，用 ATR 止損
```

**優點：** 動態調整
**缺點：** 在下跌趨勢中，價格可能一直在 EMA 下方

### 方法 4：費波那契止損

```
公式：
  做多 SL = 61.8% 回撤位下方
  做空 SL = 61.8% 反彈位上方

為什麼用費波那契：
- 61.8% 是黃金分割位
- 跌破 61.8% = 趨勢可能反轉
- 技術分析中的關鍵位

範例（HYPE 做多 @ $71.50）：
  90d 61.8% = $50.64 → 太遠了
  30d 61.8% = $56.93 → 也太遠
  → 費波那契止損適合長期持倉，不適合短線
```

**優點：** 有技術依據
**缺點：** 在短線交易中，61.8% 通常太遠

### 方法 5：固定比例止損

```
公式：
  做多 SL = 入場價 × (1 - N%)
  做空 SL = 入場價 × (1 + N%)

N = 3% ~ 5%（推薦 3% 用於 5x 槓桿，5% 用於 3x 槓桿）

範例（HYPE 做多 @ $71.50，5x 槓桿）：
  3% SL = $71.50 × 0.97 = $69.36
  5% SL = $71.50 × 0.95 = $67.93

範例（SOL 做多 @ $73.14，13x 槓桿）：
  3% SL = $73.14 × 0.97 = $70.95  ← 13x 槓桿下 = -39% 虧損
  → 13x 槓桿需要更緊的 SL
```

**優點：** 簡單
**缺點：** 沒有考慮市場波動率

### 方法 6：結構止損（MSS 相關）

```
公式：
  做多 SL = 前一個 HL（更高低點）下方
  做空 SL = 前一個 LH（更低高點）上方

為什麼用結構止損：
- 結構被破壞 = 趨勢可能反轉
- 這是最嚴格的止損方式
- 只在結構確認時使用

範例（SOL 做多 @ $73.14）：
  最近 HL = $73.39（12天前）
  SL = $73.39 - 0.5% = $73.02
  → 但價格現在 $72.27 已經低於 HL
  → 結構止損已經觸發了！
```

**優點：** 最有技術依據
**缺點：** 有時太嚴格，容易被掃

## 二、移動止損（Trailing Stop）

### 什麼是移動止損？

```
傳統 SL：固定價格，不隨價格變動
移動止損：SL 隨價格「有利方向」移動，鎖定利潤

核心邏輯：
1. 初始 SL 設寬一點（給價格波動空間）
2. 價格上漲 → SL 上移（鎖定利潤）
3. 價格下跌 → SL 不動（保護本金）
4. 只上移不下移（多頭）；只下移不上移（空頭）
```

### 移動止損 vs 傳統 SL

```
傳統 SL（固定）：
  入場 $73.14，SL $72.32（固定）
  → 價格漲到 $75，SL 還在 $72.32
  → 價格跌回 $72.5，被掃掉（虧損）

移動止損（追蹤）：
  入場 $73.14，初始 SL $71.91（3x ATR）
  → 價格漲到 $74，SL 移到 $73.18（EMA20 - 2x ATR）
  → 價格漲到 $75，SL 移到 $74.18（新高 - 2x ATR）
  → 價格跌回 $74.20，SL 觸發（但已鎖定利潤）
```

### 移動止損的三種方式

#### 方式 1：ATR 追蹤止損（推薦）

```
公式：
  多頭：SL = 最高價 - N × ATR(14)
  空頭：SL = 最低價 + N × ATR(14)

N = 2.0 ~ 3.0（推薦 2.0）

規則：
- 每次價格創新高 → 計算新的 SL
- 如果新 SL > 舊 SL → 上移 SL
- 如果新 SL < 舊 SL → 不動（只上移不下移）

範例（SOL 做多 @ $73.14，ATR = $0.41）：
  初始 SL = $73.14 - 3 × $0.41 = $71.91

  價格漲到 $74.00：
    新 SL = $74.00 - 2 × $0.41 = $73.18
    $73.18 > $71.91 → SL 上移到 $73.18 ✅

  價格漲到 $75.00：
    新 SL = $75.00 - 2 × $0.41 = $74.18
    $74.18 > $73.18 → SL 上移到 $74.18 ✅

  價格跌到 $74.50：
    新 SL = $74.50 - 2 × $0.41 = $73.68
    $73.68 < $74.18 → SL 不動（只上移不下移）✅

  價格繼續跌到 $74.18：
    SL 觸發 → 出場
    獲利 = $74.18 - $73.14 = +$1.04（+1.4%）
```

#### 方式 2：EMA 追蹤止損

```
公式：
  多頭：SL = EMA20（或 EMA50）
  做空：SL = EMA20（或 EMA50）

規則：
- 價格 > EMA20 → 持有
- 價格 < EMA20 → 出場

優點：簡單，無腦跟隨
缺點：EMA 是滯後指標，可能出場太晚
```

#### 方式 3：波段高低點追蹤止損

```
公式：
  多頭：SL = 最近波段低點 - 緩衝
  做空：SL = 最近波段高點 + 緩衝

規則：
- 每次價格創新高 → 找新的波段低點 → 設 SL
- 如果價格跌破波段低點 → 出場

優點：有技術依據
缺點：需要主觀判斷波段
```

### 移動止損的觸發條件

```
移動止損什麼時候該啟動？

方案 A：立即啟動（推薦）
- 入場後立即設移動止損
- 初始 SL 設寬（3x ATR）
- 獲利後 SL 逐漸收緊（2x ATR → 1.5x ATR）

方案 B：獲利後啟動
- 先用固定 SL（2x ATR）
- 獲利超過 5% 後，切換到移動止損
- 這樣可以給價格更多空間

方案 C：關鍵位啟動
- 先用固定 SL
- 價格突破關鍵位（如 EMA20）後，切換到移動止損
```

### 移動止損的注意事項

```
✅ 該做的：
- 只上移不下移（多頭）
- 初始 SL 要寬（3x ATR）
- 獲利後逐漸收緊（2x → 1.5x ATR）
- 配合 TP 使用（TP1 平倉後，SL 移到入場價）

❌ 不該做的：
- 不要頻繁移動 SL（容易被掃）
- 不要把 SL 設太緊（1x ATR 太容易被掃）
- 不要在下移趨勢中上移 SL
- 不要因為「覺得會漲」就不設 SL
```

## 三、TP（獲利）的五種設定方式

### 方法 1：固定比例 TP

```
公式：
  TP1 = 入場價 × (1 + 12%)  ← 7 次翻倍目標
  TP2 = 入場價 × (1 + 20%)
  TP3 = 入場價 × (1 + 30%)

平倉策略：
  TP1 平 30%
  TP2 平 40%
  TP3 平 30%
```

### 方法 2：EMA/阻力位 TP

```
公式：
  TP1 = EMA20（短期目標）
  TP2 = 前高（中期目標）
  TP3 = 1.5-2 倍入場距離（長期目標）

範例（HYPE 做多 @ $71.50）：
  TP1 = EMA20 = $73.10 → +2.2%
  TP2 = 30d 高點 = $76.54 → +7.1%
  TP3 = $71.50 + ($76.54 - $71.50) × 1.5 = $79.06 → +10.6%
```

### 方法 3：費波那契 TP

```
公式：
  TP1 = 38.2% 回撤位（從低點算起）
  TP2 = 50% 回撤位
  TP3 = 61.8% 回撤位

注意：TP 用的費波那契跟 SL 不同
  SL 是從高點往下算（回撤）
  TP 是從低點往上算（延伸）

範例（HYPE 做多 @ $71.50）：
  低點 = $34.64（90d）
  高點 = $76.54（90d）
  區間 = $41.90
  TP1 = $71.50 + $41.90 × 0.382 = $87.51 → +22.4%
  TP2 = $71.50 + $41.90 × 0.500 = $92.45 → +29.3%
  → 太遠了，不適合短線
```

### 方法 4：R:R 固定比例

```
公式：
  TP = 入場價 ± (SL 距離 × R:R)

R:R = 1:2 或 1:3

範例（HYPE 做多 @ $71.50，SL $69.85）：
  SL 距離 = $71.50 - $69.85 = $1.65
  TP1 = $71.50 + $1.65 × 2 = $74.80 → +4.6%
  TP2 = $71.50 + $1.65 × 3 = $76.45 → +6.9%
```

### 方法 5：信號反轉 TP

```
公式：
  當出現反轉信號時出場

反轉信號：
- RSI 超買（> 70）→ 做多出場
- RSI 超賣（< 30）→ 做空出場
- 看跌/看漲型態出現
- MSS 反轉
```

## 四、不同策略下的 TP/SL 推薦

### 策略 A：趨勢跟隨（大趨勢 + 短線）

```
場景：長期趨勢明確，順勢入場
槓桿：3-5x

初始 SL：3x ATR（寬鬆，因為趨勢明確）
移動止損：獲利 5% 後啟動，2x ATR 追蹤
TP1：+12%（7 次翻倍目標）→ 平 30%
TP2：+20% → 平 40%
TP3：+30% → 平 30%

R:R：1:2 ~ 1:3
```

### 策略 B：盤整區間交易

```
場景：價格在支撐/阻力之間來回
槓桿：3x

SL：1x ATR（緊，因為區間有限）
TP：對面（做多 → 阻力位，做空 → 支撐位）
移動止損：不適用（區間交易不用追蹤）

平倉：全部平倉
R:R：1:1.5 ~ 1:2
```

### 策略 C：突破交易

```
場景：價格突破盤整區間
槓桿：5x

SL：突破位下方 1x ATR
移動止損：突破後啟動，2x ATR 追蹤
TP：1-2 倍區間高度

平倉：TP1 平 50%，TP2 平 50%
R:R：1:2 ~ 1:3
```

### 策略 D：逆勢交易（反彈/回調）

```
場景：在下降趨勢中做多反彈，或上升趨勢中做空回調
槓桿：3x（逆勢要降低槓桿）

SL：1.5x ATR（較緊，因為逆勢風險大）
移動止損：不適用（逆勢不用追蹤）
TP：EMA20 或 38.2% 回撤位

平倉：全部平倉（不要貪心）
R:R：1:1.5 ~ 1:2
```

### 策略 E：MSS 確認後入場

```
場景：等結構轉換確認後入場
槓桿：5x

SL：MSS 確認位下方（前高變支撐 / 前低變阻力）
移動止損：MSS 確認後啟動，2x ATR 追蹤
TP：前高 / 前低

平倉：TP1 平 50%，TP2 平 50%
R:R：1:2 ~ 1:3
```

## 五、TP/SL 設定檢查清單

```
每次設定 TP/SL 前，問自己：

[ ] 1. SL 有技術依據嗎？（ATR / 波段低點 / EMA / 費波那契）
[ ] 2. SL 距離合理嗎？（不要太緊也不要太寬）
[ ] 3. TP 有技術依據嗎？（阻力位 / EMA / 固定比例）
[ ] 4. R:R 至少 1:2 嗎？
[ ] 5. 槓桿與 SL 匹配嗎？
     - 3x → SL 3-5%
     - 5x → SL 2-3%
     - 10x → SL 1-2%
     - 13x → SL 0.5-1%（不推薦）
[ ] 6. 考慮到手續費嗎？（TP 要 > 手續費 × 2）
[ ] 7. 考慮到大盤方向嗎？（BTC 下跌時，TP 要保守）
[ ] 8. 有移動止損計劃嗎？（初始 SL → 獲利後收緊）
```

## 六、常見錯誤

| 錯誤 | 後果 | 正確做法 |
|------|------|---------|
| SL 隨便設 | 被掃或大虧 | 用 ATR / 波段低點 |
| SL 太緊 | 被正常波動掃掉 | 至少 1.5x ATR |
| SL 太寬 | 虧損太大 | 不超過 3x ATR |
| TP 太遠 | 達不到 | 設在阻力位附近 |
| TP 太近 | 賺太少 | R:R 至少 1:2 |
| 不設 SL | 爆倉 | 永遠設 SL |
| 移動 SL 往下 | 被掃 | 只上移不下移（多頭） |
| 全部平倉 | 錯過大行情 | 分批平倉 |
| 沒有移動止損 | 利潤回吐 | 獲利後啟動追蹤 |
