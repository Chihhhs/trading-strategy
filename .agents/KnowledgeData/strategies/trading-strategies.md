## 一、趨勢跟隨策略 (Trend Following)

### 1. 移動平均線交叉策略

**基本版：**
- 買入：短期 MA 上穿長期 MA
- 賣出：短期 MA 下穿長期 MA
- 參數：EMA 9/21、EMA 20/50、SMA 50/200

**進階版（三重濾網）：**
```
第一濾網（週線）：只在主要趨勢方向交易
第二濾網（日線）：回調時尋找入場
第三濾網（小時線）：精確入場時機
```

**實證表現：**
- ✅ Aurelius 基金、AHL 等 CTA 基金 20+ 年正收益
- ⚠️ 盤整期會產生連續虧損
- 📊 勝率通常 35-45%，但盈虧比高（2:1 以上）

**來源**：Alexander Elder《Trading for a Living》

---

### 2. 海龜交易法則 (Turtle Trading)

**入場：**
- System 1：價格突破 20 日高 → 買入
- System 2：價格突破 55 日高 → 買入

**倉位管理：**
```
單位大小 = 帳戶的 1% / (N × 每點價值)
N = 20日 ATR
```

**加倉：**
- 每上漲 0.5N → 加一個單位
- 最多 4 個單位

**止損：**
- 每個單位止損 = 2N
- 最大風險 = 帳戶的 2%

**出場：**
- System 1：跌破 10 日低
- System 2：跌破 20 日低

**實證表現：**
- ✅ 1980 年代海龜學員年均回報 80%+
- ⚠️ 需要嚴格紀律，回撤可達 60%
- 📊 核心理念：趨勢 + 嚴格風控 = 長期盈利

**來源**：Curtis Faith《Way of the Turtle》

---

### 3. Donchian Channel Breakout

```
買入：價格 > 過去 n 日最高價
賣出：價格 < 過去 n 日最低價
```

- 海龜法則的基礎
- 適合加密貨幣（趨勢性強）
- 參數：n = 20（日線）、n = 55（週線）

---

### 4. SuperTrend 趨勢策略

```
買入：SuperTrend 由空翻多
賣出：SuperTrend 由多翻空
參數：ATR週期=10, 乘數=3
```

**加密貨幣優化參數：**
- 4H 時間框架：ATR=10, Multiplier=2-3
- 日線：ATR=10, Multiplier=3-4

## 二、均值回歸策略 (Mean Reversion)

### 5. 布林通道回歸策略

```
買入：價格觸及下軌 + RSI < 30
賣出：價格觸及中軌或上軌
止損：價格跌破下軌 - 1×ATR
```

**適用條件：**
- ADX < 25（無明顯趨勢）
- RSI 在 20-80 之間波動
- 價格沒有重大基本面變化

**不適用：**
- 強趨勢市場（價格會沿布林軌道運行）
- 重大新聞事件期間

---

### 6. RSI 背離策略

```
看多背離：價格創新低，RSI 未創新高
→ 買入信號

看空背離：價格創新高，RSI 未創新高
→ 賣出信號
```

**確認信號：**
- RSI 背離 + MACD 背離 = 強信號
- 在支撐/阻力位的背離 = 更強信號
- 時間框架越大，信號越可靠

---

### 7. VWAP 回歸策略

```
買入：價格下跌至 VWAP 下方 1-2%
賣出：價格回歸 VWAP
止損：價格跌破 VWAP 下方 3%
```

**適用：**日內交易，機構資金活躍的時段

---

## 三、動量策略 (Momentum)

### 8. 動量排序策略

```
1. 選擇 50 個標的
2. 計算 12-1 個月動量（排除最近一個月）
3. 買入前 20% 最強動量
4. 賣出後 20% 最弱動量
5. 每月再平衡
```

**實證表現：**
- ✅ Jegadeesh & Titman (1993)：動量效應顯著
- ⚠️ 2009 年動量策略崩潰（momentum crash）
- 📊 長期年化超額收益 4-6%

**來源**：學術文獻廣泛驗證

---

### 9. 突破動量策略

```
條件：
1. 價格突破關鍵阻力位
2. 成交量 > 平均量的 150%
3. RSI > 50
4. ADX 上升

入場：突破確認後
止損：突破位下方 1×ATR
目標：突破距離的 2-3 倍
```

---

### 10. 相對強弱策略 (Relative Strength)

```
買入：相對 BTC/ETH 強度上升的標的
賣出：相對強度下降的標的
```

**加密貨幣應用：**
- ALT/BTC 比值上升 → 資金流入 altcoin → 買入 altcoin
- ALT/BTC 比值下降 → 資金回流 BTC → 賣出 altcoin

## 四、波段交易策略 (Swing Trading)

### 11. ABCD 形態

```
A → B：上漲/下跌段
B → C：回調（0.618-0.786 Fibonacci）
C → D：與 A→B 等長
```

**入場：**C 點確認後（D 點入場）
**止損：**D 點下方
**目標：**D 點 + AB 長度

---

### 12. 頭部/底部形態

| 形態 | 信號 | 目標 |
|------|------|------|
| 雙底 (W) | 看多 | 頸線 + 形態高度 |
| 雙頂 (M) | 看空 | 頸線 - 形態高度 |
| 頭肩底 | 看多 | 頸線 + 頭部深度 |
| 頭肩頂 | 看空 | 頸線 - 頭部深度 |
| 上升三角形 | 看多 | 突破 + 三角形高度 |
| 下降三角形 | 看空 | 突破 - 三角形高度 |
| 旗形 | 延續 | 旗杆長度 |
| 楔形 | 反轉 | 楔形最寬處 |

**驗證**：✅ 100+ 年驗證，所有技術分析教材核心內容

---

### 13. Elliott Wave（艾略特波浪）

```
推動浪：5 浪結構（1-2-3-4-5）
調整浪：3 浪結構（A-B-C）

規則：
- 浪2不能超過浪1起點
- 浪3不能是最短的推動浪
- 浪4不能進入浪1價格區間
```

- **驗證**：⚠️ 主觀性強，不同分析師數浪結果不同
- **交易用途**：波浪計數 + Fibonacci = 入场/出场區域

## 五、日內交易策略 (Day Trading)

### 14. 開盤區間突破 (Opening Range Breakout, ORB)

```
1. 記錄開盤後 15-30 分鐘的最高/最低
2. 突破上軌 → 買入
3. 突破下軌 → 賣出
4. 止損：反向突破
```

---

### 15. VWAP 日內策略

```
做多條件：
- 價格在 VWAP 上方
- 回調至 VWAP 附近
- 成交量支持
- RSI > 50

做空條件：相反
```

---

### 16. EMA 9/21 日內交叉

```
買入：EMA9 上穿 EMA21 + 價格在 EMA200 上方
賣出：EMA9 下穿 EMA21
止損：近期波段低點
```

**適用：**1-15 分鐘圖表，高流動性標的

## 六、加密貨幣專用策略

### 17. 資金費率套利 (Funding Rate Arbitrage)

```
當資金費率 > 0.05%（8h）：
- 做空 perp + 買入 spot
- 收取資金費率

當資金費率 < -0.05%（8h）：
- 做多 perp + 賣出 spot
- 收取資金費率
```

**實證表現：**
- ✅ 年化 15-30%（在牛市高資金費率期間）
- ⚠️ 需要雙向持倉，佔用保證金
- ⚠️ 極端行情下可能虧損

---

### 18. Altcoin Season 策略

```
Bitcoin Dominance (BTC.D) 上升 → 持有 BTC
BTC.D 下降 → 輪換到 ALT

判斷 Alt Season：
1. BTC.D 從高點下降 > 5%
2. USDT Dominance 下降（資金流入風險資產）
3. ALT/BTC 配對普遍上漲
4. 中小市值幣種漲幅 > BTC
```

---

### 19. 恐懼貪婪指數策略

```
極度恐懼 (0-20)：分批買入
恐懼 (20-40)：持續買入
中性 (40-60)：持有
貪婪 (60-80)：開始獲利了結
極度貪婪 (80-100)：大量賣出
```

**驗證**：✅ 長期來看，恐懼時買入、貪婪時賣出獲利顯著

## 七、機構級策略

### 21. 風險平價 (Risk Parity)

```
每個資產對組合的風險貢獻相等
倉位 ∝ 1 / 波動率

高波動資產 → 小倉位
低波動資產 → 大倉位
```

**實證表現：**
- ✅ Bridgewater All Weather 基金長期有效
- ⚠️ 需要頻繁再平衡
- 📊 夏普比率通常優於等權重

---

### 22. 統計套利 (Stat Arb)

```
1. 找到相關的配對資產
2. 計算價差的 Z-Score
3. Z-Score > 2 → 做空價差
4. Z-Score < -2 → 做多價差
5. Z-Score 回歸 0 → 平倉
```

**加密貨幣應用：**
- BTC/ETH 比值交易
- 同板塊代幣配對（如 L1 之間）
- 需要持續監控相關性變化

## 八、策略評估框架

### 關鍵績效指標 (KPI)

| 指標 | 計算 | 好的標準 |
|------|------|---------|
| 勝率 | 盈利交易/總交易 | > 50%（趨勢策略可接受 35-45%） |
| 盈虧比 | 平均盈利/平均虧損 | > 2.0 |
| 期望值 | (勝率×平均盈利) - (敗率×平均虧損) > 0 | 正數 |
| 最大回撤 | 高峰到谷底的損失 | < 20%（激進可接受 30%） |
| 夏普比率 | (報酬-無風險利率)/標準差 | > 1.0（優秀 > 2.0） |
| 索提諾比率 | (報酬-無風險利率)/下行標準差 | > 1.5 |
| 卡爾馬比率 | 年化報酬/最大回撤 | > 1.0 |
| 交易次數 | 每月/每年交易次數 | 避免過度交易 |
| 最大連續虧損 | 最長虧損序列 | 心理承受範圍內 |

### 回測注意事項

1. **前視偏差**：確保不使用未來數據
2. **手續費**：包含 maker/taker 手續費（0.02-0.1%）
3. **滑價**：市價單通常有 0.1-0.5% 滑價
4. **樣本外測試**：至少保留 30% 數據做樣本外測試
5. **穩健性測試**：微調參數後效果是否穩定
6. **不同市場週期**：牛市、熊市、盤整期都要測試
