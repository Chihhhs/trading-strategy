# 動態 TP/SL 策略

> 來源：專業交易員經驗、風險管理文獻
> 驗證狀態：✅ 廣泛驗證

---

## 一、什麼是動態 TP/SL

**定義：** 根據市場價格變動和浮盈狀況，動態調整止損（SL）和止盈（TP）位置，而不是固定不動。

**核心邏輯：**
```
固定 TP/SL：設定後不變，可能過早被洗出或回吐利潤
動態 TP/SL：隨價格移動，保護利潤同時讓利潤奔跑
```

**與固定 TP/SL 的區別：**

| 維度 | 固定 TP/SL | 動態 TP/SL |
|------|-----------|-----------|
| 止損位置 | 固定不變 | 隨價格移動 |
| 止盈位置 | 固定不變 | 可分批調整 |
| 風險控制 | 可能過早被洗出 | 隨浮盈增加而保本 |
| 利潤最大化 | 可能過早獲利了結 | 讓利潤奔跑 |
| 心理壓力 | 高（怕被洗出） | 低（有保本機制） |

## 二、動態 TP/SL 核心原則

### 原則 1：只往有利方向移動止損

```
多頭：止損只上移，不下移
空頭：止損只下移，不上移

原因：
- 保護已實現的利潤
- 避免過早被震盪洗出
- 讓趨勢充分發展
```

### 原則 2：浮盈保本機制

| 浮盈比例 | 行動 | 說明 |
|---------|------|------|
| 0-5% | 保持原始 SL | 給價格波動空間 |
| 5-10% | SL 移至入場價 + 2% | 確保小幅獲利 |
| 10-15% | SL 移至入場價 | 完全保本 |
| 15%+ | SL 移至浮盈 50% 位置 | 保護大部分利潤 |

### 原則 3：分批獲利

```
不要一次全部平倉！

建議分批：
- TP1（+3-5%）：平倉 30%
- TP2（+8-10%）：平倉 40%
- TP3（+15%+）：平倉 30%

原因：
- 避免過早平倉錯過大行情
- 降低心理壓力
- 平衡風險和報酬
```

### 原則 4：趨勢追蹤止損

```
方法 A：ATR 追蹤止損
- 多頭：止損 = 最高價 - N×ATR
- 空頭：止損 = 最低價 + N×ATR
- N 通常為 2-3

方法 B：均線追蹤止損
- 多頭：價格跌破 EMA 20 → 減倉 50%
- 多頭：價格跌破 EMA 50 → 全部平倉
- 空頭：價格突破 EMA 20 → 減倉 50%
- 空頭：價格突破 EMA 50 → 全部回補

方法 C：結構追蹤止損
- 多頭：價格跌破前一個波段低點 → 出場
- 空頭：價格突破前一個波段高點 → 出場
```

### 原則 5：趨勢減弱信號

```
以下信號出現時，考慮減倉或出場：

1. ADX 從高點回落（趨勢強度減弱）
2. 成交量持續下降（參與度降低）
3. 出現反轉 K 線形態（如射擊之星、錘子線）
4. RSI 背離（價格創新高但 RSI 未創新高）
5. MACD 柱狀圖縮小（動能減弱）
```

### 原則 6：時間止損

```
如果持有時間過長而無明顯方向：

- 持有超過 2 週無明顯趨勢 → 減倉 50%
- 持有超過 1 個月 → 重新評估策略
- 持有期間有更好的機會 → 考慮換倉

原因：
- 資金效率
- 避免機會成本
- 趨勢可能已經結束
```

## 三、動態 TP/SL 實作範例

### 範例 1：SUI 空倉動態調整

```
初始倉位：Short 22.70 SUI @ $0.80
初始 SL：$0.85（入場價 + 6%）

動態調整：
1. 價格跌至 $0.78（浮盈 2.5%）
   → SL 保持 $0.85（未達 5% 門檻）

2. 價格跌至 $0.76（浮盈 5%）
   → SL 移至 $0.776（現價 + 2%，保本）

3. 價格跌至 $0.72（浮盈 10%）
   → SL 移至 $0.80（完全保本）

4. 價格跌至 $0.68（浮盈 15%）
   → SL 移至 $0.74（保護 50% 浮盈）

5. 價格跌至 $0.64（浮盈 20%）
   → SL 移至 $0.70（保護 60% 浮盈）

TP 設定：
- TP1：$0.748（-5%）→ 平倉 30%
- TP2：$0.708（-10%）→ 平倉 40%
- TP3：$0.680（-15%）→ 平倉 30%
```

### 範例 2：HYPE 多倉動態調整

```
初始倉位：Long 1.0 HYPE @ $65.00（假設回調到 EMA20 入場）
初始 SL：$61.00（入場價 - 6%）

動態調整：
1. 價格漲至 $68.00（浮盈 4.6%）
   → SL 保持 $61.00（未達 5% 門檻）

2. 價格漲至 $68.25（浮盈 5%）
   → SL 移至 $66.30（現價 - 2.8%，保本）

3. 價格漲至 $71.50（浮盈 10%）
   → SL 移至 $65.00（完全保本）

4. 價格漲至 $74.75（浮盈 15%）
   → SL 移至 $68.25（保護 50% 浮盈）

5. 價格漲至 $78.00（浮盈 20%）
   → SL 移至 $71.50（保護 60% 浮盈）

TP 設定：
- TP1：$68.25（+5%）→ 平倉 30%
- TP2：$71.50（+10%）→ 平倉 40%
- TP3：$74.75（+15%）→ 平倉 30%
```

## 四、動態 TP/SL 的注意事項

### 常見錯誤

```
❌ 過早移動止損：浮盈 2% 就急著保本，容易被震盪洗出
❌ 過度貪心：已經浮盈 20% 還不移動止損，最後回吐
❌ 頻繁調整：每根 K 線都調整，容易被短期波動干擾
❌ 忽略趨勢：在強趨勢中過早獲利了結
```

### 最佳實踐

```
✅ 設定明確的調整規則（如浮盈 5% 開始保本）
✅ 使用 ATR 或均線作為客觀調整依據
✅ 分批獲利，不要一次全平
✅ 記錄每次調整，定期回顧效果
✅ 根據市場環境調整規則（波動大時用更寬的止損）
```

## 五、動態 TP/SL 檢查清單

### 每次調整前檢查

```
□ 目前浮盈是多少？
□ 是否達到調整門檻（5%、10%、15%）？
□ 趨勢是否仍然完整？
□ 是否有反轉信號？
□ 新的止損位置是否合理？
□ 是否需要分批獲利？
```

### 調整後記錄

```
□ 調整時間
□ 調整前的 SL/TP
□ 調整後的 SL/TP
□ 調整原因（浮盈保本/趨勢追蹤/反轉信號）
□ 當前浮盈百分比
```
