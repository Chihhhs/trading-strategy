# 大趨勢策略（Macro Trend Following）

> 來源：經典交易文獻、對沖基金實務、加密市場特性
> 驗證狀態：✅ 廣泛驗證 | ⚠️ 部分驗證

---

## 一、什麼是「大趨勢策略」

**定義：** 辨識並跟隨市場的主要趨勢方向（數週到數月），忽略短期波動，在趨勢確立後入場，在趨勢反轉前出場。

**核心理念：**
```
趨勢是你的朋友（Trend is your friend）
讓利潤奔跑，截斷虧損（Let profits run, cut losses short）
不是預測市場，而是反應市場
```

**與短線交易的區別：**

| 維度 | 大趨勢策略 | 短線交易 |
|------|-----------|---------|
| 時間框架 | 日線/週線 | 1H/4H/15m |
| 持有時間 | 數週-數月 | 數小時-數天 |
| 交易頻率 | 每月 1-3 次 | 每天多次 |
| 手續費影響 | 低 | 高 |
| 心理壓力 | 低（不常看盤） | 高 |
| 勝率 | 30-40% | 50-60% |
| 盈虧比 | 3:1 - 5:1 | 1.5:1 - 2:1 |

## 二、大趨勢策略的核心要素

### 1. 趨勢辨識

**多時間框架分析（MTF）：**
```
週線：判斷主要趨勢方向
日線：尋找入場區域
4H/1H：精確入場時機
```

**趨勢判斷工具：**
| 工具 | 用法 | 驗證 |
|------|------|------|
| EMA 20/50/200 | 多頭排列 = 上升趨勢 | ✅ |
| ADX > 25 | 強趨勢 | ✅ |
| 高低點結構 | 更高高點+更高低點 = 上升 | ✅ |
| 趨勢線 | 連接兩個以上低點/高點 | ✅ |
| Ichimoku | 雲的方向 + 價格位置 | ✅ |

### 2. 入場時機

**順勢入場（趨勢確立後）：**
```
1. 等待趨勢確認（突破關鍵位/均線交叉）
2. 不回調不追高
3. 在回調到支撐/均線時入場
4. 分批入場（30% + 40% + 30%）
```

**反轉入場（趨勢可能反轉時）：**
```
1. 價格到達極端值（RSI 超買/超賣）
2. 出現背離信號
3. 成交量異常放大
4. 謹慎使用，需要更多確認
```

### 3. 出場策略

**趨勢追蹤止損：**
```
方法 A：ATR 追蹤止損
- 止損 = 最高價 - 3×ATR（多頭）
- 只上移不下移

方法 B：均線追蹤止損
- 價格跌破 EMA 20 → 出場
- 價格跌破 EMA 50 → 全部出場

方法 C：結構追蹤止損
- 價格跌破前一個波段低點 → 出場
```

**目標獲利：**
```
方法 A：固定 R:R
- 風險報酬比 1:3 或 1:5

方法 B：趨勢結束信號
- ADX 從高點回落
- 價格跌破關鍵均線
- 出現反轉 K 線形態
```

### 4. 倉位管理

**風險控制：**
```
單筆風險：帳戶的 1-2%
總風險暴露：不超過 60%
最大連續虧損：3 次後暫停交易
```

**加倉策略：**
```
金字塔加倉：
- 初始倉位：30%
- 趨勢確認後加倉：40%
- 突破關鍵位後加倉：30%

條件：
- 只在盈利時加倉
- 每次加倉量遞減
- 加倉後移動止損
```

## 三、大趨勢策略的具體方法

### 方法 1：均線交叉 + ADX 過濾

```
入場條件（做多）：
1. EMA 20 上穿 EMA 50
2. ADX > 25（確認趨勢強度）
3. 價格 > EMA 200（長期趨勢向上）
4. 回調到 EMA 20 附近入場

出場條件：
1. EMA 20 下穿 EMA 50
2. 或價格跌破 EMA 50
3. 或 ATR 追蹤止損被觸發
```

### 方法 2：高低點結構突破

```
入場條件（做多）：
1. 價格突破前一個波段高點
2. 回調不破前一個波段低點
3. 成交量確認（突破時放量）

出場條件：
1. 價格跌破前一個波段低點
2. 或出現更低低點 + 更低高點
```

### 方法 3：Ichimoku 雲策略

```
入場條件（做多）：
1. 價格在雲上方
2. Tenkan 上穿 Kijun
3. 雲為綠色（看多）
4. Chikou Span 在價格上方

出場條件：
1. 價格跌破雲
2. Tenkan 下穿 Kijun
```

### 方法 4：超級趨勢（SuperTrend）

```
入場條件（做多）：
1. SuperTrend 由紅轉綠
2. 價格在 SuperTrend 上方
3. 確認成交量支持

出場條件：
1. SuperTrend 由綠轉紅
2. 價格跌破 SuperTrend
```

## 四、加密貨幣大趨勢的特殊考量

### 1. 24/7 市場

```
優勢：
- 沒有跳空缺口
- 趨勢更連續
- 隨時可以出場

挑戰：
- 需要自動化監控
- 夜間波動可能很大
- 需要設定好止損單
```

### 2. 高波動性

```
調整：
- 使用更寬的止損（3-5x ATR）
- 降低倉位大小
- 使用較慢的均線（EMA 50/200）
- 避免在重大新聞事件前入場
```

### 3. 相關性

```
BTC 主導：
- BTC 漲 → Altcoin 通常跟漲
- BTC 跌 → Altcoin 通常跟跌更多
- 做多做空都應該先看 BTC 方向

相關性變化：
- 牛市初期：BTC 主導
- 牛市中期：Altcoin 輪動
- 牛市末期：Altcoin 補漲
- 熊市：全部下跌
```

### 4. 鏈上指標（加密特有）

```
看多信號：
- 交易所流出增加（持有意願上升）
- 長期持有者增持
- MVRV Z-Score < 0（低估）
- 穩定幣流入增加

看空信號：
- 交易所流入增加（準備賣出）
- 長期持有者減持
- MVRV Z-Score > 7（高估）
- 穩定幣流出增加
```

## 五、大趨勢策略的績效預期

### 歷史回測數據

| 策略 | 年化報酬 | 最大回撤 | 夏普比率 | 驗證 |
|------|---------|---------|---------|------|
| 均線交叉 + ADX | 15-25% | 20-30% | 1.0-1.5 | ✅ |
| 高低點突破 | 20-35% | 25-35% | 1.2-1.8 | ✅ |
| Ichimoku | 15-20% | 15-25% | 0.8-1.2 | ⚠️ |
| SuperTrend | 10-20% | 20-30% | 0.7-1.0 | ⚠️ |

### 關鍵成功因素

```
1. 紀律：嚴格執行策略，不隨意更改
2. 耐心：等待好的入場機會，不頻繁交易
3. 風險管理：永遠不超過 2% 單筆風險
4. 趨勢辨識：正確判斷趨勢方向
5. 出場策略：不要過早獲利了結
```
