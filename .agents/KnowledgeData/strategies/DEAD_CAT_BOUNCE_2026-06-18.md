# 假上漲（死貓反彈）分析報告 2026-06-18

## 📊 背景

用戶在 2026-06-18 發現：
1. 策略在上漲趨勢中虧損
2. 持有期感覺太長
3. 全部止損後，發現上漲趨勢「很奇怪」

經過深度分析，發現：**這不是真正的上漲趨勢，而是死貓反彈（Dead Cat Bounce）**。

---

## 🔍 死貓反彈的定義

**死貓反彈（Dead Cat Bounce）**：
- 資產價格大幅下跌後出現的反彈
- 反彈幅度通常為跌幅的 1/3 到 2/3
- 反彈結束後繼續下跌
- 名稱來自「即使是一隻死貓，從高處掉下來也會彈一下」

## 💡 為什麼是假上漲？

### 原因 1：長期趨勢未改變

- SMA50 遠高於當前價格
- 180d 高點到當前價格的跌幅 > 50%
- 一次的技術性反彈無法改變長期趨勢

### 原因 2：反彈量能不足

- 健康的上漲需要「上漲放量、下跌縮量」
- 當前是「上漲缩量、下跌放量」
- 表示買方力量不足以推動持續上漲

### 原因 3：反彈時間太短

- 死貓反彈通常持續 5-15 個交易日
- 當前反彈已持續約 5-8 個交易日
- 之後價格通常會繼續下跌

### 原因 4：基本面未改善

- 市場整體仍處於下跌趨勢
- 沒有明確的利好消息推動反轉
- 機構投資者仍在撤出

## 📋 教訓

1. **不要在上漲趨勢中追多** — 首先要確認這是不是「真正的上漲」
2. **死貓反彈是正常的** — 大幅下跌後的反彈不代表反轉
3. **SMA 排列很重要** — SMA7 < SMA20 < SMA50 = 空頭排列，不要做多
4. **量價關係是關鍵** — 上漲放量才是健康上漲
5. **反彈幅度 < 跌幅的 50%** — 這是死貓反彈，不是反轉

## 🔄 改善策略

### 加入「死貓反彈過濾器」

```python
def is_dead_cat_bounce(closes, highs, lows, sma20, sma50):
    """判斷是否為死貓反彈"""
    
    current = closes[-1]
    
    # 條件 1：SMA 空頭排列
    if sma20[-1] >= sma50[-1]:
        return False  # 不是空頭排列，不是死貓反彈
    
    # 條件 2：價格從低點反彈 10-30%
    recent_low = min(lows[-30:])
    bounce_pct = (current / recent_low - 1) * 100
    if bounce_pct < 5:
        return False  # 反彈太少，不是死貓反彈
    if bounce_pct > 40:
        return False  # 反彈太多，可能已經反轉
    
    # 條件 3：從高點跌幅 > 30%
    high_180 = max(highs)
    drop_pct = (current / high_180 - 1) * 100
    if drop_pct > -20:
        return False  # 跌幅不夠大，不是死貓反彈
    
    # 條件 4：最近 3 天中有 2 天以上下跌
    down_days = sum(1 for i in range(-3, 0) if closes[i] < closes[i-1])
    if down_days >= 2:
        return True  # 反彈可能已經結束
    
    return True
```

| 死貓反彈結束 | 增大做空倉位 |
| 真正反轉 | 需要等待 SMA 多頭排列確認 |
