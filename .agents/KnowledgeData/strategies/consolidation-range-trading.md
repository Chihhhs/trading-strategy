# 盤整期短線交易策略

> 更新：2026-06-16
> 背景：BTC -2.6% 帶領全部 Altcoin 下跌，市場方向不明，進入盤整

---

## 一、什麼是盤整？

```
盤整 = 價格在明確的支撐/阻力之間來回波動，沒有明確方向

上升趨勢 → 盤整 → 繼續上升或反轉下降
下降趨勢 → 盤整 → 繼續下降或反轉上升

盤整期的特徵：
- 價格在支撐/阻力之間來回
- RSI 在 30-70 之間波動
- 成交量沒有明顯方向
- 沒有 HH+HL 或 LH+LL 的明確結構
```

## 二、盤整期的三種短線策略

### 策略 A：區間交易（Range Trading）

```
在支撐位做多，在阻力位做空

操作步驟：
1. 找出最近的支撐位和阻力位
2. 在支撐位附近做多
3. 在阻力位附近做空
4. 止損設在支撐/阻力位之外

入場條件：
- 價格到達支撐位 + RSI < 40 → 做多
- 價格到達阻力位 + RSI > 60 → 做空

出場條件：
- TP：對面（做多 → 阻力位，做空 → 支撐位）
- SL：支撐/阻力位之外 2-3%
```

**SOL 盤整區間（2026-06-16）：**
```
支撐位：$72.25（最近低點）
阻力位：$73.84（最近高點）

做多：價格到 $72.25-72.50 + RSI < 40
做空：價格到 $73.50-73.84 + RSI > 60
SL：做多 $71.50 / 做空 $74.50
TP：做多 $73.50 / 做空 $72.50
```

### 策略 B：突破交易（Breakout Trading）

```
等價格突破盤整區間，順勢入場

操作步驟：
1. 確認盤整區間（支撐 + 阻力）
2. 等價格突破阻力位（做多）或跌破支撐位（做空）
3. 確認突破有效（收盤站穩 + 放量）
4. 入場

入場條件：
- 日線收盤 > 阻力位 + 成交量 > 平均 → 做多
- 日線收盤 < 支撐位 + 成交量 > 平均 → 做空

出場條件：
- TP：1-2 倍區間高度
- SL：突破位下方/上方
```

**SOL 突破區間：**
```
支撐：$72.25
阻力：$73.84
區間高度：$1.59

做多：收盤 > $73.84 + 放量
  TP：$73.84 + $1.59 = $75.43
  SL：$73.84

做空：收盤 < $72.25 + 放量
  TP：$72.25 - $1.59 = $70.66
  SL：$72.25
```

### 策略 C：MSS 等待（結構轉換等待）

```
最保守的策略：等 MSS 確認再入場

操作步驟：
1. 確認市場在盤整
2. 等價格突破前高（MSS 看多）或跌破前低（MSS 看空）
3. 等回測確認（HL 或 LH）
4. 入場

→ 這是最安全的策略
→ 但需要耐心，可能等很久
```

## 三、盤整期的紀律

### ✅ 該做的

| 規則 | 說明 |
|------|------|
| 降低倉位 | 盤整期不確定性大，倉位減半 |
| 縮小 TP | 目標不要太大，小賺就跑 |
| 嚴格 SL | 盤整可能突然變方向，一定要設止損 |
| 不追高殺低 | 在區間中間不入場 |
| 耐心等待 | 盤整期大部分時間應該空手 |

### ❌ 不該做的

| 錯誤 | 後果 |
|------|------|
| 在區間中間入場 | 被來回打臉 |
| 不設止損 | 突然突破 = 大虧 |
| 重倉 | 盤整期虧損時間長 |
| 頻繁交易 | 手續費侵蝕利潤 |
| 逆勢操作 | 盤整期沒有趨勢可順 |

## 四、盤整期的入場檢查清單

```
[ ] 1. 確認市場在盤整（HH+LL 或 LH+HL）
[ ] 2. 畫出支撐位和阻力位
[ ] 3. 價格在支撐/阻力附近？（不是中間）
[ ] 4. RSI 支持？（做多 < 40，做空 > 60）
[ ] 5. 成交量支持？（突破時放量）
[ ] 6. 長期趨勢支持？（至少不逆勢）
[ ] 7. 設好 SL 和 TP

如果 5/7 以上確認 → 小倉位入場
如果 < 5/7 → 空手等待
```
