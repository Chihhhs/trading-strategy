## 二、建議增加的指標

### 類別 1：趨勢強度（建議加入）

| 指標 | 說明 | 建議權重 | 優先級 |
|------|------|---------|--------|
| **ADX** | 趨勢強度量化（>25 強趨勢） | ±8 | ⭐⭐⭐ 高 |
| **趨勢持續時間** | 趨勢已持續多少根 K 線 | ±5 | ⭐⭐ 中 |
| **趨勢一致性** | 多時間框架趨勢是否一致 | ±7 | ⭐⭐⭐ 高 |

**理由：** 目前只有趨勢方向（BULLISH/BEARISH），但沒有趨勢強度。ADX 可以量化趨勢強度，避免在弱趨勢中交易。

### 類別 2：波動率（建議加入）

| 指標 | 說明 | 建議權重 | 優先級 |
|------|------|---------|--------|
| **波動率狀態** | 波動率是擴張還是收縮 | ±6 | ⭐⭐⭐ 高 |
| **波動率百分位** | 當前波動率在歷史中的位置 | ±5 | ⭐⭐ 中 |
| **布林通道寬度** | 通道寬度反映波動率 | ±4 | ⭐⭐ 中 |

**理由：** 波動率影響倉位大小和止損距離。高波動率時應該降低倉位，低波動率時可以加倉。

### 類別 3：市場情緒（建議加入）

| 指標 | 說明 | 建議權重 | 優先級 |
|------|------|---------|--------|
| **恐懼貪婪指數** | 市場整體情緒 | ±8 | ⭐⭐⭐ 高 |
| **資金費率** | 衍生品市場情緒 | ±6 | ⭐⭐⭐ 高 |
| **社交媒體情緒** | Twitter/Reddit 情緒 | ±5 | ⭐⭐ 中 |
| **未平倉合約變化** | 市場參與度 | ±5 | ⭐⭐ 中 |

**理由：** 情緒指標是反向指標，極度恐懼時買入，極度貪婪時賣出。目前系統缺少這類指標。

### 類別 4：鏈上數據（建議加入）

| 指標 | 說明 | 建議權重 | 優先級 |
|------|------|---------|--------|
| **交易所淨流量** | 資金流入/流出交易所 | ±8 | ⭐⭐⭐ 高 |
| **長期持有者持倉** | 聰明錢的行為 | ±7 | ⭐⭐⭐ 高 |
| **穩定幣供應量** | 場外資金準備入場 | ±6 | ⭐⭐ 中 |
| **MVRV Z-Score** | 估值指標 | ±7 | ⭐⭐⭐ 高 |
| **NVT Ratio** | 網絡價值/交易量 | ±5 | ⭐⭐ 中 |

**理由：** 鏈上數據是加密市場特有的指標，反映真實的持有者行為和資金流向。

### 類別 5：宏觀經濟（建議加入）

| 指標 | 說明 | 建議權重 | 優先級 |
|------|------|---------|--------|
| **美元指數（DXY）** | 美元強弱影響加密 | ±6 | ⭐⭐⭐ 高 |
| **美國利率政策** | 影響風險資產 | ±5 | ⭐⭐ 中 |
| **美股走勢** | 風險偏好指標 | ±5 | ⭐⭐ 中 |
| **黃金走勢** | 避險需求 | ±4 | ⭐ 低 |

**理由：** 宏觀經濟影響加密市場整體方向，特別是在大趨勢策略中。

### 類別 6：市場結構（建議加入）

| 指標 | 說明 | 建議權重 | 優先級 |
|------|------|---------|--------|
| **市場主導地位** | BTC.D 變化 | ±6 | ⭐⭐⭐ 高 |
| **Altcoin Season Index** | Alt 季節 | ±5 | ⭐⭐ 中 |
| **相關性** | 幣種間相關性 | ±4 | ⭐ 低 |

**理由：** 市場結構影響資金流向，BTC.D 上升時資金流向 BTC，下降時流向 Alt。

## 三、建議的 v4 評分架構

### 核心層（必選，目前已有）

| 因素 | 權重 | 說明 |
|------|------|------|
| 趨勢方向 | ±12 | BULLISH/BEARISH |
| RSI | ±8 | 超買/超賣 |
| MACD | ±8 | 動能 |
| 成交量 | ±6 | 量價配合 |
| 支撐阻力 | ±8 | 位置 |
| K線形態 | ±4 | 形態 |
| 背離 | ±8 | 背離信號 |
| Liquidity Sweep | ±8 | 清掃信號 |
| Order Flow | ±8 | 訂單流 |
| VWAP/VP | ±8 | 成交量分佈 |

### 增強層（建議加入）

| 因素 | 權重 | 說明 |
|------|------|------|
| ADX（趨勢強度） | ±6 | 趨勢強度 |
| 波動率狀態 | ±6 | 波動率擴張/收縮 |
| 恐懼貪婪指數 | ±6 | 市場情緒 |
| 資金費率 | ±4 | 衍生品情緒 |
| 交易所淨流量 | ±6 | 鏈上資金流向 |
| MVRV Z-Score | ±6 | 估值 |
| DXY | ±4 | 美元強弱 |
| BTC.D | ±4 | 市場主導地位 |

### 總分計算

```
總分 = 核心層（±80）+ 增強層（±40）= ±120

標準化到 ±100：
最終分數 = 總分 / 1.2
```
