## 一、利率與加密市場

### 1. 美國聯邦基金利率

**影響機制：**
```
升息 → 美元走強 → 風險資產吸引力下降 → BTC 跌
降息 → 美元走弱 → 風險資產吸引力上升 → BTC 漲
```

**歷史相關性：**
| 利率週期 | BTC 表現 | 驗證 |
|---------|---------|------|
| 2022 升息週期 | -64%（熊市） | ✅ |
| 2023 暫停升息 | +150%（牛市） | ✅ |
| 2024 降息預期 | +120% | ✅ |

**交易規則：**
```
FOMC 會議前 2 天：
- 減少倉位或觀望
- 波動加大，容易止損

FOMC 會議後：
- 鷹派結果 → 做空或觀望
- 鴿派結果 → 做多
- 符合預期 → 延續原趨勢
```

**驗證**：✅ 2022-2024 年高度相關

---

### 2. 美元指數（DXY）

```
DXY 上升（美元走強）→ BTC 傾向下跌
DXY 下降（美元走弱）→ BTC 傾向上漲

相關性：約 -0.6 到 -0.7（負相關）
```

**交易平台：** TradingView 上搜尋 "DXY"

**交易策略：**
```
DXY 突破關鍵阻力 → 考慮減少加密倉位
DXY 跌破關鍵支撐 → 考慮增加加密倉位
DXY 與 BTC 同步上漲 → 特殊情況（如避險需求）
```

---

### 3. 美國國債收益率

```
10Y 收益率上升 → 無風險利率提高 → 風險資產吸引力下降 → BTC 跌
10Y 收益率下降 → 無風險利率降低 → 風險資產吸引力上升 → BTC 漲
```

**當前（2026）：**
- 10Y 收益率約 4.2-4.5%
- 如果超過 5%，可能對 BTC 形成壓力
- 如果低於 3.5%，可能利好 BTC

## 二、通膨指標

### 4. CPI（消費者物價指數）

| CPI 結果 | 市場解讀 | BTC 反應 | 驗證 |
|---------|---------|---------|------|
| CPI > 預期 | 通膨升溫 | 跌（升息預期） | ✅ |
| CPI < 預期 | 通膨緩解 | 漲（降息預期） | ✅ |
| CPI = 預期 | 中性 | 波動小 | ✅ |

**發布時間：** 每月第二個週三 20:30 UTC

**交易策略：**
```
CPI 前：
- 減少倉位
- 等待數據公布

CPI 後：
- 比預期低 → 做多
- 比預期高 → 做空或觀望
- 與預期相近 → 延續原趨勢
```

---

### 5. PPI（生產者物價指數）

```
PPI > 預期 → 成本推動通膨 → BTC 跌
PPI < 預期 → 通膨壓力減輕 → BTC 漲

PPI 是 CPI 的領先指標
```

**發布時間：** 每月第二個週二 20:30 UTC

## 四、量化寬鬆/緊縮

### 8. QE（量化寬鬆）vs QT（量化緊縮）

```
QE（印鈔）：
- 央行買入資產 → 市場流動性增加
- 風險資產價格上升
- BTC 通常大漲

QT（縮表）：
- 央行賣出資產或不續買 → 市場流動性減少
- 風險資產價格下跌
- BTC 通常下跌
```

**歷史案例：**
| 政策 | 時間 | BTC 反應 |
|------|------|---------|
| COVID QE | 2020-2021 | +1000% |
| 開始 QT | 2022 | -64% |
| 降息預期 | 2023 | +150% |

**驗證**：✅ 高度相關

## 五、地緣政治風險指數

### 9. Geopolitical Risk Index（GPR）

```
GPR 上升 → 不確定性增加 → 避險需求上升
→ 黃金、BTC 上漲

GPR 下降 → 風險偏好回升
→ 股票、高風險資產上漲
```

**交易策略：**
```
GPR 突然上升：
1. 減少槓桿
2. 設定更緊的 SL
3. 等待市場穩定
4. 如果 BTC 沒跌甚至漲 → 強勢信號 → 可以做多

GPR 持續高位：
1. BTC 可能成為避險資產
2. 考慮增加 BTC 倉位
3. 減少 altcoin 倉位（alt 風險更高）
```

## 六、傳統市場相關性

### 10. 美股（S&P 500、NASDAQ）

```
2022-2024 相關性：約 +0.6 到 +0.8（正相關）

美股上漲 → BTC 傾向上漲（風險偏好）
美股下跌 → BTC 傾向下跌（風險厭惡）

例外情況：
- 避險需求時，BTC 可能與美股脫鉤
- 美元走弱時，BTC 可能獨自上漲
```

**交易策略：**
```
美股開盤時間（14:30-21:00 UTC）：
- 波動最大
- 美股方向影響 BTC

美股收盤後：
- 波動減小
- BTC 可能有獨立走勢
```

---

### 11. 黃金

```
黃金上漲 → 避險需求 → BTC 可能跟隨
黃金下跌 → 風險偏好 → BTC 可能獨自上漲

相關性：約 +0.3 到 +0.5（中等正相關）
```

## 八、宏觀環境評估框架

### 牛熊市場判斷

| 指標 | 牛市信號 | 熊市信號 |
|------|---------|---------|
| 利率 | 降息/暫停升息 | 升息 |
| DXY | 下降 | 上升 |
| CPI | 下降或穩定 | 上升 |
| QE/QT | QE 或暫停 QT | QT |
| 美股 | 上漲 | 下跌 |
| 黃金 | 穩定或下跌 | 上漲（避險） |
| 市場情緒 | 貪婪但不過度 | 恐懼但不過度 |

**使用方法：**
- 7 個指標中 >= 5 個看多 → 牛市
- 7 個指標中 >= 5 個看空 → 熊市
- 混合信號 → 盤整/不確定
