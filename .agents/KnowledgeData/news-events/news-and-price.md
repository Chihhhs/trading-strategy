## 一、地緣政治事件與加密市場

### 1. 戰爭與衝突

**歷史案例：**

| 事件 | 時間 | BTC 反應 | 持續時間 | 驗證 |
|------|------|---------|---------|------|
| 俄烏戰爭爆發 | 2022-02-24 | 當日 -8%，1週後 +15% | 短期恐慌後反彈 | ✅ |
| 以巴衝突 | 2023-10-07 | 當日 +5%，1月後 +40% | 持續上漲 | ✅ |
| 中東緊張（伊朗/以色列） | 2024-04 | 當日 -5%，2週後創新高 | V型反轉 | ✅ |
| 台海緊張 | 2022-08 | 當日 -3%，1週後恢復 | 短期影響 | ⚠️ |

**戰爭利好的邏輯鏈：**
```
戰爭爆發
  → 政府增加軍費支出
    → 央行印鈔或舉債
      → 通膨上升 / 法幣貶值
        → 避險資產需求增加（BTC、黃金）
          → BTC 上漲

戰爭爆發
  → 資本管制加強
    → 資金外逃需求
      → 加密貨幣成為管道
        → BTC/穩定幣需求增加

戰爭爆發
  → 供應鏈中斷
    → 經濟不確定性
    → 去中心化資產吸引力上升
      → 加密市場資金流入
```

**交易策略：**
| 階段 | 市場反應 | 策略 |
|------|---------|------|
| 戰爭爆發初期 | 恐慌拋售 | 觀望，等待企穩 |
| 恐慌消退（1-3天） | 開始反彈 | 分批買入 |
| 戰爭持續 | 避險需求增加 | 持有或加倉 |
| 戰爭結束 | 風險偏好回升 | 獲利了結 |

**驗證**：✅ 2022-2024 年多次驗證

**監管事件交易策略：**
```
利空事件：
1. 事件發生 → 不要恐慌拋售
2. 等待 1-3 天 → 市場消化消息

利好事件：
1. 事件發生 → 不要追高
2. 等待回調 → 逢低買入
3. 利好通常有持續性 → 可以持有較久
```

### 3. 宏觀經濟事件

**利率決策（FOMC）：**
```
鷹派（升息/不降息）→ BTC 短期下跌 5-15%
鴿派（降息/暗示降息）→ BTC 短期上漲 5-20%
符合預期 → 波動較小

關鍵規則：
- 會議前 1-2 天：波動加大，減少倉位
- 會議當天：等待結果，不要提前下注
- 會議後：根據結果方向交易
```

**CPI/PPI 數據：**
```
CPI > 預期 → 通膨擔憂 → 可能升息 → BTC 跌
CPI < 預期 → 通膨緩解 → 可能降息 → BTC 漲
CPI = 預期 → 波動較小

發布時間：每月第二個週三 20:30 UTC
```

**非農就業數據（NFP）：**
```
NFP > 預期 → 經濟強勁 → 可能升息 → BTC 跌
NFP < 預期 → 經濟放緩 → 可能降息 → BTC 漲

發布時間：每月第一個週五 20:30 UTC
```

**驗證**：✅ 2022-2024 年多次驗證

## 三、事件驅動交易策略

### 6. 「買謠言，賣新聞」策略

```
預期利好事件：
1. 事件公布前 1-2 周 → 市場預期升溫 → 價格上漲
2. 事件公布當天 → 「賣新聞」→ 價格回調
3. 策略：事件前買入，事件後賣出

預期利空事件：
1. 事件公布前 1-2 周 → 市場恐慌 → 價格下跌
2. 事件公布當天 → 「買謠言」→ 價格反彈
3. 策略：事件前賣出，事件後買入
```

**經典案例：**
- 比特幣 ETF 通過：公布前漲 40%，公布後回調 15%
- FOMC 會議：會議前波動，會議後方向明確
- 減半事件：減半前 6 個月開始漲，減半後繼續漲

**驗證**：✅ 廣泛驗證

### 7. 黑天鵝事件應對

**定義：** 極端罕見、影響巨大、無法預測的事件

**歷史黑天鵝：**
| 事件 | BTC 反應 | 恢復時間 |
|------|---------|---------|
| COVID 崩盤 2020-03 | -50% | 3 個月 |
| LUNA 崩潰 2022-05 | -40% | 進入熊市 |
| FTX 崩潰 2022-11 | -25% | 2 個月 |
| 3AC 崩潰 2022-06 | -30% | 3 個月 |

**黑天鵝應對策略：**
```
預防：
1. 永遠不要全倉
2. 設定硬止損
3. 保留現金儲備
4. 分散投資

應對：
1. 不要恐慌拋售
2. 等待市場穩定（1-3 天）
3. 評估基本面是否改變
4. 如果基本面沒變 → 逢低買入
5. 如果基本面改變 → 止損離場
```

**驗證**：✅ 多次驗證

### 9. 新聞情緒分析

| 指標 | 來源 | 解讀 |
|------|------|------|
| 恐懼貪婪指數 | alternative.me | <20 恐懼（買入機會），>80 貪婪（賣出機會） |

**情緒交易策略：**
```
極度恐懼（0-20）：
- 市場過度悲觀
- 考慮分批買入
- 設定寬止損

極度貪婪（80-100）：
- 市場過度樂觀
- 考慮分批賣出
- 減少倉位

中性（40-60）：
- 等待方向明確
- 減少交易頻率
```

**驗證**：✅ 長期驗證有效

## 五、事件交易檢查清單

### 交易前檢查

```
□ 是否有重大新聞事件即將發生？
□ 事件的影響方向是什麼？
□ 市場是否已經price in？
□ 我的倉位是否過度暴露？
□ 是否設定了止損？
□ 是否有應急計劃？
```

### 事件發生時

```
□ 不要立即反應
□ 等待市場消化消息（1-30 分鐘）
□ 確認消息的真實性
□ 評估對基本面的影響
□ 根據策略執行交易
```

### 事件發生後

```
□ 市場是否如預期反應？
□ 是否需要調整倉位？
□ 是否需要移動止損？
□ 記錄這次事件的教訓
```
